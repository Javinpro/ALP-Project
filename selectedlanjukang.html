What to do?

Aktivitas Pagi Hari

Pagi hari di Pulau Lanjukang dimulai dengan pemandangan matahari terbit yang memukau. Para pengunjung bisa memanfaatkan waktu pagi untuk:

  • Jalan-Jalan pagi
  • Meditasi dan Yoga
  • Fotografi
Serunya Snorkling di sore hari

Setelah menikmati pagi yang tenang, saatnya untuk merasakan petualangan bawah laut di sore hari. Perairan disekitar Pulau Lanjukang terkenal dengan keindahan terumbu karangnya. Aktivitas disini sangat direkomendasikan karena:

  • Terumbu karang yang indah
  • Air Laut yang jernih
  • Keamanan dan kenyamanan
Malam hari dengan api unggun

Saat matahari terbenam dan malam mulai menyelimuti pulau, suasana menjadi semakin magis dengan kegiatan api unggun. Malam api unggun bisa diisi dengan kegiatan:

  • Api unggun
  • Barbeque
  • Cerita dan musik
